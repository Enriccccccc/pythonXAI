## 🎲 一、擲骰子小程式（隨機數）

👉 功能：電腦幫你擲骰子很多次

### 做了什麼？

1️⃣ 問你要擲幾次骰子
2️⃣ 電腦每次隨機出 1～6 的數字
3️⃣ 把結果存起來
4️⃣ 一次全部顯示出來

### 重點觀念：

✅ random.randint(1,6) = 隨機出 1～6
✅ 用 list（清單）存結果
✅ 用 for 迴圈重複做很多次

---

## 🤖 二、跟 AI 聊天（最簡單版）

👉 功能：你輸入一句話，AI 回你

### 做了什麼？

1️⃣ 讀取 API 金鑰（密碼）
2️⃣ 建立 AI 客戶端
3️⃣ 一直問你要說什麼
4️⃣ 把你的話傳給 AI
5️⃣ 印出 AI 回答

### 重點觀念：

✅ input() 讓使用者輸入
✅ while True 一直聊天
✅ quit 可以離開
