# Python 超簡單筆記 🐍

## 1️⃣ 註解 = 給自己看的說明

```python
# 單行
"""
多行
"""
```

---

## 2️⃣ 常用資料

- 整數 → 3
- 小數 → 3.14
- 文字 → "hi"
- 對錯 → True / False

---

## 3️⃣ 顯示東西

```python
print("嗨")
```

---

## 4️⃣ 算數學

```python
a = 10
b = 3

a + b  # 加
a - b  # 減
a * b  # 乘
a / b  # 除
a // b # 只取整數
a % b  # 取餘數
a ** b # 次方
```

---

## 5️⃣ 文字玩法

```python
"hi" + "!"    # 連起來 → hi!
"hi" * 3      # 重複 → hihihi
f"我叫 {name}"  # 放變數
len("hi")     # 幾個字 → 2
```

---

## 6️⃣ 看資料型態

```python
type(10)   # 整數
type("hi") # 文字
```

---

## 7️⃣ 轉換資料

```python
int("3")   # 文字 → 整數
float(3)   # 整數 → 小數
str(3)     # 整數 → 文字
bool(0)    # 0 → False, 其他 → True
```

---

## 8️⃣ 請人輸入

```python
name = input("你叫什麼名字？")
print("嗨 " + name)
```

---

## 9️⃣ 判斷（if）

```python
if password == "1234":
    print("歡迎")
else:
    print("密碼錯")
```

---

## 🔟 邏輯簡單版

- and → 都對才對
- or → 有一個對就對
- not → 相反

```python
True and False # False
True or False  # True
not True       # False
```

---

## 1️⃣1️⃣ 小數學應用：圓面積

```python
r = int(input("半徑:"))
area = 3.14 * r * r
print(area)
```

>       "文字" → 文字
>       True/False → 對錯
