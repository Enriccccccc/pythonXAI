## 🛒 一、購物籃小程式（用 Streamlit）

👉 這是一個「可以加商品的購物車」

重點概念：

### ✅ 購物籃用「字典」存資料

像這樣：

```python
cart = {"蘋果": 2, "香蕉": 1}
```

意思是：
👉 蘋果有2個
👉 香蕉有1個

---

### ➕ 加入商品

如果商品已經有了 → 數量 +1
如果沒有 → 新增商品

就像：

🧺 已經有蘋果 → 再加一顆
🧺 沒有橘子 → 加進去

---

### 🗑 清空 & 刪除

• 可以清空整個購物籃
• 也可以刪掉某一個商品

---

## ➗ 二、算術指定運算子（快速計算）

這些是「懶人寫法」😆

| 寫法      | 意思      |
| --------- | --------- |
| a += b    | a = a + b |
| a -= b    | a = a - b |
| a \*= b   | a = a × b |
| a /= b    | a = a ÷ b |
| a //= b   | 整數除法  |
| a %= b    | 取餘數    |
| a \*\*= b | 次方      |

### 🌟 小例子：

```python
a = 10
b = 3
a += b
```

👉 a 變成 13

---

## 🎯 三、猜數字遊戲（電腦想一個數字）

電腦先想一個秘密數字（0～100）

你一直猜：

👆 太小 → 提示「大一點」
👇 太大 → 提示「小一點」
🎉 猜對 → 遊戲結束

還會縮小範圍，讓你更容易猜中！

---

## 🔁 四、while 迴圈（一直做，直到停）

### 範例：

```python
i = 0
while i < 5:
    print(i)
    i += 1
```

意思是：

👉 從 0 印到 4

---

### ⛔ break（強制停下來）

```python
if i == 3:
    break
```

👉 當 i 變成 3 就不做了！

---

## 🔂 五、for 迴圈（數數用）

```python
for i in range(1,6):
    print(i)
```

會印出：

1 2 3 4 5

👉 range 就像「數數機」

---

## 📦 六、字典（dictionary）

像「成績表」📊

### 建立字典：

```python
scores = {"A":90, "B":80}
```

意思是：

A → 90分
B → 80分

---

### 📖 讀資料

```python
scores["A"]
```

👉 得到 90

---

### ➕ 新增

```python
scores["C"] = 70
```

---

### ✏ 修改

```python
scores["A"] = 100
```

---

### 🗑 刪除

```python
scores.pop("C")
```

---

### 🔍 檢查有沒有

```python
"A" in scores
```

👉 有 → True
👉 沒有 → False

---

### 📏 看有幾個資料

```python
len(scores)
```

---

## 📚 七、用字典存很多成績 + 算平均

像這樣：

```python
scores = {
  "math": [],
  "english": []
}
```

每一科放很多分數：

👉 math 放10個分數
👉 english 放10個分數

然後：

✔ 加起來
✔ 除以10
✔ 算平均
